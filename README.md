# Nutrify - AI Food Analyzer

Nutrify is a smart food analysis application that uses AI to analyze your meals, provide health scores, and track your dietary history.

## Features
- **AI Food Analysis**: Upload an image of your meal to get instant nutritional insights.
- **Health Scoring**: Get a health score based on the ingredients.
- **Dietary History**: Track your past meals and scores.
- **User Profiles**: Manage your personal details for personalized analysis.
- **Premium UI**: Experience a beautiful, modern, and responsive interface.

## Setup
1.  Clone the repository.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Set up environment variables in a `.env` file:
    ```
    GOOGLE_API_KEY=your_api_key
    SECRET_KEY=your_secret_key
    ```
4.  Run the application:
    ```bash
    python app.py
    ```

## Technologies
- Frontend: HTML, CSS (Premium Glassmorphism Design), JavaScript
- Backend: Flask, Python
- AI: Google Gemini API
- Database: SQLite
